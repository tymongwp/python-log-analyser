# Python Log Analyzer

A simple Python tool that reads a log file and counts failed login attempts. 
---

## What it does

This program scans a log file and checks how many times a **failed login** appears.

* Finds lines with "failed login"
* Counts how many times it happens
* Displays the total

---

## Features

* Simple and beginner-friendly
* Reads any text-based log file
* Case-insensitive search
* Basic error handling

---

## How to run

Run the script:

```bash
python loganalyser.py
```
